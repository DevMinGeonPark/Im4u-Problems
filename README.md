# Im4u 문제 풀이

Im4u 영재원에서 C 언어와 알고리즘(올림피아드)을 강의하며 직접 만들고 풀어 본 문제와 강의노트 모음입니다. 수업 난이도에 맞춰 정리한 자료라 거창한 라이브러리는 없고, 학생들에게 설명하기 좋게 풀어 쓴 C 소스가 대부분입니다.

강사로 일한 흔적을 남겨 두려고 모아 둔 저장소입니다. 입력은 `INPUT.txt`로 받고 출력은 `OUTPUT.txt`로 내보내는 경시대회식 파일 입출력 형태를 그대로 따랐습니다.

## 구성

난이도와 주제별로 폴더를 나눠 두었습니다.

- `초급/` — 조건문, 중첩 반복문, 함수, 파일 입출력 등 입문 단계 실습
- `중급하/`, `중급상/` — 레벨 테스트와 중급 문제 풀이
- `기초알고리즘/수학/` — 유클리드 호제법, 소수 판별(밀러-라빈) 같은 수론 기초
- `new-Ajit/`, `new-경시대비-레벨테스트/` — 경시대회 대비 문제와 레벨 테스트

## 메모

학생 수준에 맞춰 같은 문제를 여러 방식으로 풀어 본 코드도 섞여 있습니다. 정답 코드라기보다 수업에서 함께 따라가며 설명한 흔적에 가깝습니다.
