# Im4u Problems

A collection of problems and lecture notes I wrote and worked through while teaching C and competitive algorithms (olympiad) at the Im4u gifted academy. The material is pitched at the class level, so there are no heavy libraries here, just C source written to be easy to explain to students.

I keep this repository as a record of my teaching work. The code follows the contest-style file I/O convention: input comes from `INPUT.txt` and output goes to `OUTPUT.txt`.

## Layout

Folders are split by difficulty and topic.

- `초급/` (beginner) — intro exercises on conditionals, nested loops, functions, and file I/O
- `중급하/`, `중급상/` (lower/upper intermediate) — level tests and intermediate problems
- `기초알고리즘/수학/` (math basics) — number theory fundamentals like the Euclidean algorithm and Miller-Rabin primality testing
- `new-Ajit/`, `new-경시대비-레벨테스트/` — contest preparation problems and level tests

## Note

Some problems appear solved more than one way, matched to where the students were. These are less polished answer keys and more a trace of what I walked through with the class.
